从SqlSessionFactory中获取SqlSession

  • 顾名思义，有了SqlSessionFactory，我们就可以从中获取SqlSession。
  • SqlSession 完全包含了面向数据库执行 SQL 命令所需的所有方法。
  • 可以通过 SqlSession 实例来直接执行已映射的 SQL 语句。
