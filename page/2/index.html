Java 包装类详解

发表于 2017-09-15 | 分类于 Java
本文字数： 1.5k | 阅读时长 ≈ 1 分钟

简介

  • Java语言是一个面向对象的语言，但是Java中的基本数据类型却是不面向对象的为了解决这个不足，在设计类时为每个基本数据类型设计了一个对应的类进行代表，这样八个和基本数据类型对应的类统称为包装类(Wrapper Class)，有些地方也翻译为外覆类或数据类型类。
  • 包装类均位于Java.lang包，包装类和基本数据类型的对应关系如下表所示：
基本数据类型 包装类
byte Byte
boolean Boolean
short Short
char Character
int Integer
long Long
float Float
double Double
